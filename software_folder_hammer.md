# FolderHammer 1.2.1
FolderHammer は空フォルダ削除ツールです。
削除されるフォルダを事前に確認でき、実際に削除するかどうか事前に設定することが可能です。
ごみ箱へ削除する場合、一番上のフォルダをごみ箱に入れます。その際、下位のフォルダやファイルは削除しません。
